課程主要內容

  • HTML 學習如何將文字，文件及圖片轉換成適合瀏覽器閱讀的格式。
  • CSS 可以讓你縮短網站開發時間，更容易維護，更好的易用性，也能吸引更多使用者。
  • 課程會以專案開發方式進行，實戰是最好的練習。
